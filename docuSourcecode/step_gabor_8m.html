Detailed Description

Step function for the Gabor layer.

Author
Frederik Beuth (beuth.nosp@m.@cs..nosp@m.tu-ch.nosp@m.emni.nosp@m.tz.de)

Small helper function to filter an image with a Gabor kernel. The Gabor kernel is not calculated here for speed reasons, but should be rather provided to this function. It is intended to define the Gabor kernel beforehand (see function gaborKernel.m).

  • This version processes a grayscale image.
